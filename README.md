# 模擬紙本日曆的「電子日曆」

![]( image/t_0.gif)
![]( image/c0_2.png)

✔長按+拖曳：撕日曆  

✔從畫面最左邊的邊緣向右滑：叫出日曆選單  

✔從畫面最上方的邊緣向下滑：可以回到上一頁日曆  

![]( image/cal_re3.png)

## 可以像APP一樣安裝到手機主畫面。
